## About Me

KXAND是 Quicsand 的意思，而后者来源和读音是 quick sand。
这个名字源自于某个小说的某个叫流砂的角色，我把这个名字扔进谷歌翻译后把结果搅拌了一下，然后朋友们都这么叫于是就成了我的网名。
我会用本账号搞一些乱七八糟的事情（大概），所以通常这个是我的活跃账号。
